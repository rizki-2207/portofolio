# Personal Portfolio - Rizki Ananda

Ini adalah proyek situs web portofolio pribadi yang dibangun untuk menampilkan keahlian, riwayat pendidikan, pengalaman, dan sertifikasi.

## Teknologi yang Digunakan
- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS v4
- **Bahasa:** TypeScript
- **Ikon:** Lucide React
- **Deployment:** Vercel

## Fitur Utama
- 🌗 **Dark Mode Toggle:** Tema terang dan gelap yang terintegrasi penuh.
- 📱 **Responsif:** Desain mobile-first yang rapi di segala ukuran layar.
- 🚀 **Performa Cepat:** Dioptimalkan menggunakan komponen bawaan Next.js dan Vercel Speed Insights.

## Cara Menjalankan Proyek (Lokal)
1. Pastikan Node.js sudah terinstal.
2. Buka terminal di folder proyek ini dan jalankan `npm install` untuk mengunduh semua dependensi.
3. Jalankan `npm run dev`.
4. Buka `http://localhost:3000` di browser Anda.
